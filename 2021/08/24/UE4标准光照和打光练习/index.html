Part1 标准光照环境设置

​

​

Directional Light 2.14 + Skylight 1.0 = 3.14

turn off Armosphric Sun Light

All image effect off (like Vignette, Lens flare…)

White Balance should be neutralized: Set to 7000 // UE4 color temperature default value = 6500, which is biased to blue

roughness limit should raised up to 1.0

​

then check the 18% gray which should be (128, 128, 128). Use the Exposure adjustment to control it.

Real world Lighting Ratio = 16: 1 (direct+indirect : indirect)
